Settings & Sync

Enter a GitHub Personal Access Token to sync data across devices. Only needs the gist scope — no other permissions required. Your data is stored in a secret (private) Gist.
Important: use a classic token (not fine-grained) — fine-grained tokens don't support the Gist API.

How far back to look for food entries when suggesting triggers.

1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Click Generate new token (classic)
3. Set a name (e.g. "symptom-tracker"), tick only gist, click Generate
4. Copy the token starting with ghp_ and paste it above
